== Description ==

This plugin can add a text ads in widget or theme .

"Buy Pro Version":

`http://marketwp.ir/product/parsi-text-ads-pro/`


The features of this plugin can be listed as follows:

* Add text ads with title , description , Domain URL , Custumer Contact
* Many color style for text ads
* Add rel follow & nofollow to link
* Add target to link
* Expired time for ads
* Status after expire
* Status of ads in admin panel ( Active Or Expired)
* Search in advertising With Result

Language Support:

* English
* Persian
* Full RTL & LTR

Send email for Translation files: alivazirinia@yahoo.com


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `Parsi-Text-ADS.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php echo do_shortcode('[parsi_text_ads]');?>` in your theme
